# 吞食天地 - Unity 复刻版

## 项目简介

这是经典红白机（FC/NES）游戏《吞食天地》的 Unity 复刻项目。

《吞食天地》是 CAPCOM 于 1989 年在红白机上发布的角色扮演游戏，以三国演义为背景，深受玩家喜爱。

## 游戏特色

- **经典 RPG 玩法**：回合制战斗系统
- **三国故事背景**：体验刘备、关羽、张飞的冒险
- **丰富的角色系统**：收集并培养三国武将
- **策略性战斗**：排兵布阵，技能搭配

## 技术栈

- **游戏引擎**：Unity 2022.3 LTS
- **编程语言**：C#
- **版本控制**：Git
- **项目管理**：GitHub

## 项目结构

```
tianshi-tiandi/
├── Assets/                    # Unity 资源文件夹
│   ├── Scripts/              # C# 脚本
│   │   ├── Core/            # 核心系统
│   │   ├── Battle/          # 战斗系统
│   │   ├── Character/       # 角色系统
│   │   ├── UI/              # 用户界面
│   │   └── Utils/           # 工具类
│   ├── Art/                  # 美术资源
│   │   ├── Sprites/         # 精灵图
│   │   ├── Animations/      # 动画文件
│   │   └── UI/              # UI 美术
│   ├── Audio/                # 音频资源
│   │   ├── BGM/             # 背景音乐
│   │   └── SFX/             # 音效
│   ├── Prefabs/              # 预制体
│   ├── Scenes/               # 场景文件
│   └── Resources/            # 动态加载资源
├── Packages/                  # Unity 包配置
├── ProjectSettings/          # 项目设置
└── README.md                  # 项目说明文档
```
